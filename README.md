# water-hill-garden-mystery-fixed

Water Hill Garden（東京都八王子市）を舞台にした周遊AR謎解きゲームのWebアプリです。  
タイトルは **「100年の庭園と、四つ葉の継承」**。

## 概要

- 9ステップ構成の謎解き進行（title / prologue / play / ending）
- 現地の看板・石碑・展示物を想定したカメラスキャン演出
- ヒント表示、回答判定、進行マップ表示
- 最終ステップ到達でエンディング表示

## 開発環境

- React + Vite
- lucide-react（UIアイコン）

## セットアップ

```bash
npm install
```

## 開発サーバー起動

```bash
npm run dev
```

## 品質確認

```bash
npm run lint
npm run build
```
