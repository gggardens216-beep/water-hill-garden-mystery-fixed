# water-hill-garden-mystery-fixed

Water Hill Garden 100年の庭園と四つ葉の継承  
東京都八王子市の複合施設「Water Hill Garden（吉田観賞魚・グリーンギャラリーガーデンズ・ガーデンズマルシェ・オコジュ）」を舞台にした、次世代型周遊AR謎解きゲームのWebアプリです。

## 公開手順（GitHub + Vercel）

1. GitHub で新しいリポジトリ（例: `mystery-app`）を作成します。
2. `App.jsx`、`package.json`、`tailwind.config.js`、`README.md` をそのリポジトリにアップロードします。
3. [Vercel](https://vercel.com/) にアクセスし、**Add New** から対象リポジトリを選択します。
4. **Deploy** を押します。

これで数分後に、一般公開用のURLが発行されます。
