# water-hill-garden-mystery-fixed

## ARミステリーゲームアプリの概要

「Water Hill Garden（吉田観賞魚・グリーンギャラリーガーデンズ・ガーデンズマルシェ・オコジュ）」を舞台にした、周遊型AR謎解きWebアプリです。  
プレイヤーは現地を巡りながら、カメラで対象物をスキャンし、9つの謎を解いて100年の物語の真相に辿り着きます。

## 主な体験

- **没入型ストーリー**: 1921年から続く庭園の記憶をたどる連続シナリオ
- **ARスキャン演出**: ブラウザカメラを利用した現地オブジェクトの解析体験
- **段階的な謎解き進行**: 各ステップで「調査 → スキャン → 解答入力 → 次の地点へ」
- **ヒント・マップ機能**: 行き詰まりを防ぐ補助導線を用意

## ゲーム進行フロー

1. タイトル画面で探索開始
2. プロローグでプレイヤー名を入力
3. 9つのポイントを順番に探索
4. 各地点でスキャン対象を解析し、解答を入力
5. 最終ステップをクリアするとエンディングへ到達
